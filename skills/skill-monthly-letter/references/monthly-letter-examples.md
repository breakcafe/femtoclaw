# 月来信示例

以下 2 封示例来信展示月来信的目标输出风格：战略视角、预算执行评估、储蓄率追踪、消费结构月度对比、下月规划。

---

## 示例一：健康月份 + 单个类别超支

```
=== 2026年2月 | 支出 ¥8432 | 预算执行 87% | 储蓄率 22% ===
Lilly，本月支出¥8432，近三月均值¥8910，低了5%——整体节奏是稳的。不过拆开看，餐饮已经连续第三个月超预算了，这不是某一周吃多了的问题，是预算本身偏低了。

本月消费全景：
  收入 ¥10810 | 支出 ¥8432 | 结余 ¥2378 | 储蓄率 22%
  Top 类别：住房租房 ¥2800（33%）| 餐饮 ¥1950（23%）| 购物 ¥1380（16%）
餐饮连续三个月维持在¥1800以上，说明这就是你的真实消费水平，预算应该跟上来。

预算执行：
  餐饮：预算 ¥1500 / 实际 ¥1950（130%）连续三月超支，预算偏低
  购物：预算 ¥1500 / 实际 ¥1380（92%）执行良好
  交通：预算 ¥600 / 实际 ¥485（81%）本月有结余
  日用品：预算 ¥800 / 实际 ¥520（65%）持续低于预算
整体来看，超支集中在餐饮一个类别，其他类别控制得不错，问题不在消费习惯，在预算分配。

**建议：**
* **餐饮和日用品的预算可以做一次对调**：餐饮三月均值¥1870，预算只有¥1500，每个月必超；日用品三月均值¥530，预算¥800，每个月都剩。总预算不变的情况下，把餐饮调到¥1800、日用品调到¥600，预算才有参考意义。
  → **行动**：请在侧边栏"预算管理"里把餐饮预算从¥1500调整到¥1800，日用品从¥800调整到¥600。

* **本月结余¥2378，可以给攒钱计划加一笔**：连续两个月结余都在¥2000以上，说明收支差是稳定的。与其让这笔钱留在活期慢慢被零花掉，不如每月固定存一个数。
  → **行动**：可以在侧边栏"攒钱计划"里选"定额存钱"模板，设置每月存¥2000，剩下的¥378留作弹性空间。

---
```

## 示例二：春节超支月

```
=== 2026年1月 | 支出 ¥15780 | 预算执行 52% | 储蓄率 -18% ===
Lilly，本月支出¥15780，近三月均值¥9200，高了71%——但先别被这个数字吓到。春节月的交通¥2400和人情往来¥1500合计¥3900，剥离掉这部分季节性支出后，日常消费¥11880，比均值高大约29%。超出的部分主要来自年货采购和节前聚餐，不是常态消费失控。

本月消费全景：
  收入 ¥13360 | 支出 ¥15780 | 结余 -¥2420 | 储蓄率 -18%
  Top 类别：购物 ¥4200（27%）| 住房租房 ¥2800（18%）| 交通 ¥2400（15%）| 餐饮 ¥2350（15%）
交通里有往返高铁¥1680和市内出行¥720，人情往来¥1500是三家亲戚的年礼，这些都是年度性支出。

预算执行：
  购物：预算 ¥1500 / 实际 ¥4200（280%）含年货¥2600，日常购物¥1600
  餐饮：预算 ¥1500 / 实际 ¥2350（157%）节前聚餐密集
  交通：预算 ¥600 / 实际 ¥2400（400%）春运往返
春节月看预算执行的绝对数没有参考意义，重点是剥离季节性支出后的日常部分：日常购物¥1600（107%）、日常餐饮约¥1800（120%），偏高但幅度可控。

**建议：**
* **明年一月提前调高预算，今年的数据就是参考线**：春节支出是确定会发生的，与其每年一月看着预算爆表，不如在十二月就把一月总预算从¥10000临时上调到¥15000，交通和人情各留¥2000额度。这样一月的预算执行率不会失真，其他月份也不受影响。
  → **行动**：可以在手机备忘录里设一个12月25日的提醒："调整一月预算"，到时候在侧边栏"预算管理"里把月度总预算临时调高。

* **二月是天然的回血月，可以锁定一个储蓄额度**：春节结束后社交支出骤降，二月的日常支出通常比均值低15-20%。上个二月你花了¥7800，如果这个月也在这个水平，收入¥13000的话可以存下¥5000以上。
  → **行动**：可以在侧边栏"攒钱计划"里选"定额存钱"模板，二月目标存¥4000——留足余量，但把回血窗口用起来。

---
```

## 关键范式

1. **标题承载 3 个 KPI**：支出绝对值 / 预算执行率% / 储蓄率%，一行看全月健康度
2. **开篇隔离结构性 vs 季节性**：春节/双十一/开学等月份先剥离一次性支出，再看日常部分
3. **预算执行段用统一格式**：「{类别}：预算 ¥X / 实际 ¥Y（Z%）{短评}」，逐类别展示
4. **建议是战略级的**：跨月预算调整、季节窗口利用、储蓄额度锁定——不是"下周少吃两顿外卖"
5. **三月均值替代四周均值**：月来信的时间锚点是"近三月"，不是"近四周"
6. **月度对比用具体数字**：「餐饮从上月¥1200升到¥1950（+63%）」而非「餐饮有所增加」
